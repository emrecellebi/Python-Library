## Yorum Satırları
Pyhton dilinde yorum satırı nasıl oluşturulur.

```python
# Single Line Comment (Tek Satır Yorum)

"""
Multi Line Comment (Çok Satır Yorum)
"""
```

## String Concatenate
Python dilinde iki diziyi bir leş tirmek için `+` karakteri kullanılır.

```python
str = "Concat" + "enate";
```
